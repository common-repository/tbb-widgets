3 advanced widgets, which replace the default Recent Posts, Recent Comments and Text widget.

== Description ==

3 advanced widgets for your WordPress!

= TBB's Recent Posts =

Features:

*   CSS class.
*   Query variables, like tags, categories and post type.
*   Thumbnail customization.
*   Content customization.

= TBB's Recent Comments =

Features:

*   CSS class.
*   Query variables, like status, type and post type.
*   Avatar customization.
*   Content customization.

= TBB's Text =

Features:

*   CSS class.
*   PHP mode
*   HTML mode with shortcodes.

== Installation ==

1. Install The Blackest Box's Widgets either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Install Redux Framework either via the WordPress.org plugin directory, or by uploading the files to your server.
3. Activate both plugins.
4. That's it.  You're ready to go!
